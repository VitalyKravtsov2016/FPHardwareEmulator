Нужно создать приложение на Java - эмулятор аппаратуры фискального регистратора. 
Нужно сделать эмуляцию электроники: эмуляцию процессора и всех контроллеров, ФН должен реализовать протокол ФН и быть подключен в состав ФР. 
Электрическая схема и марки микросхем есть в папке doc

Электрическая схема фискального регистратора - в папке doc
В состав фр также входит ФН, фискальный накопитель. 
Протокол ФН есть в папке doc.
Программа нужна для написания тестов, 
то есть для тестирования прошивок ФР.
У программы есть GUI: 
- загрузить прошивку в формате bin
- позволяет включить или выключить
- крышка корпуса открыта/закрыта
- Рулон бумаги есть или нет
- Бумага есть или нет
- ФН подключен или нет
- окно чека, в котором показывается напечатанный чек
