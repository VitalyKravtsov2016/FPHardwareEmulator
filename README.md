# FP Hardware Emulator

Эмулятор аппаратуры фискального регистратора (ФР) на Java с графическим интерфейсом.
Предназначен для тестирования прошивок ФР без реального оборудования.

## Функциональность

### GUI

| Элемент | Описание |
|---|---|
| **Загрузить прошивку (.bin)** | Загрузка файла прошивки для тестирования |
| **Включить / Выключить** | Эмуляция питания устройства |
| **Крышка корпуса** | Переключение состояния крышки (открыта / закрыта) |
| **Рулон бумаги** | Наличие / отсутствие рулона бумаги |
| **Бумага** | Наличие / отсутствие бумаги в принтере |
| **ФН** | Подключение / отключение фискального накопителя |
| **Открыть / Закрыть смену** | Управление сменами через ФН |
| **Тестовый чек** | Печать тестового фискального чека |
| **Окно чека** | Отображение напечатанных чеков и системных сообщений |

### Регистр статуса оборудования

Байт состояния оборудования (аналог аппаратного регистра):

| Бит | Значение |
|---|---|
| Бит 0 (0x01) | Крышка открыта |
| Бит 1 (0x02) | Бумага закончилась |
| Бит 2 (0x04) | Рулон отсутствует |
| Бит 3 (0x08) | ФН не подключен |

```
src/main/java/com/fpemulator/
├── Main.java                        # Точка входа
├── hardware/

```

## Требования

- Java 17+
- Maven 3.8+

## Сборка и запуск

```bash
# Сборка
mvn package

# Запуск
java -jar target/fp-hardware-emulator-jar-with-dependencies.jar
```

## Использование

1. **Загрузите прошивку** — нажмите "Загрузить прошивку (.bin)" и выберите бинарный файл
2. **Включите устройство** — нажмите "Включить"
3. **Откройте смену** — нажмите "Открыть смену"
4. **Печатайте чеки** — нажмите "Тестовый чек"
5. **Управляйте состоянием** — изменяйте флаги крышки, бумаги, ФН в реальном времени

## Протокол ФН

Эмулятор реализует основные команды протокола ФН:

| Команда | Код | Описание |
|---|---|---|
| GET_FN_STATUS | 0x01 | Получить статус ФН |
| GET_FN_NUMBER | 0x02 | Получить заводской номер ФН |
| OPEN_SESSION | 0x10 | Открыть смену |
| CLOSE_SESSION | 0x14 | Закрыть смену |
| OPEN_RECEIPT | 0x21 | Открыть чек |
| ADD_ITEM | 0x22 | Добавить позицию в чек |
| CLOSE_RECEIPT | 0x24 | Закрыть чек |
| CANCEL_RECEIPT | 0x25 | Отменить чек |
| GET_LAST_FD | 0x30 | Получить последний фискальный документ |
